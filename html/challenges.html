Challenges

The challenges below are best worked through in order, becauase the earlier challenges help you learn how to complete the later ones.

Each challenge contains:

  • A programming task
  • An objective of what we will learn from the task
  • A description of any new programming tools that you might use to complete the task, that have not been seen in previous challenges
  • An example program

Your program might be adapted from the example, or you can create something completely different!

Web Technologies

The challenges below provide a basic introduction to 2 technologies widely used in the World Wide Web today: HTML and JavaScript.
There are many other technologies out there, but we don't have time to cover everything and the main aim is to cover programming techniques that are common to any technology.

Before you start these challenges, you need to complete your web development setup.
