Setup

We will be using Processing, which is a simplified environment for Java, a popular programming language used today.

Processing was originally designed as visual environment in whick to learn basic computer programming, but is also now widely used by artists and designers to produce professional work.

Processing should already be set up on your computer in the club. Just double click "Processing.exe" to run the tool.

If you have access to a computer at home, you can follow the download instructions here and try the examples in your own time.
