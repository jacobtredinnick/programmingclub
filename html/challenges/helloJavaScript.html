Challenge - Hello World in JavaScript

Task

Write an HTML page that makes the browser show a "Hello World!" prompt:

Objective

Write your first web page with dynamic content using JavaScript..

Tools

"Hello World" is the traditional first step to learn any programming language.
Once again you just need a text editor, a browser and to follow along with the steps.

  • Step 1 : Create the HTML file

    • Open your text editor (we can use Notepad for now) and create a new file.

  • Step 2 : Write the HTML code, with embedded JavaScript

    • Type the following text into the editor (the script tag is the embedded JavaScript):

    <!DOCTYPE HTML> <html> <body> <p>This page shows a "Hello World!" prompt as soon as it is loaded or refreshed.</p> <script> alert('Hello World!') </script> </body> </html>

    Save the file to something like "HelloJavaScript.html" in a directory that you have access to.

  • Step 3 : Open the file in your browser

    • Double click the file you just created. It should open up in your browser and look something like the image above.

  • Step 4 : Congratulations!

    • You have written and run some JavaScript to create dynamic content that not only displays information, but responds to user input.

    That's a bit more useful than only using HTML, but it's not very impressive yet. We will use it for some more interesting things in the next challenges...
