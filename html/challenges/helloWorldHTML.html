Challenge - Hello World HTML

Task

Write an HTML page that shows "Hello World!" in the browser:

Objective

Write your first web page.

Tools

"Hello World" is the traditional first step to learn any programming language.
Since this is the first challenge, you just need a text editor, a browser and to follow along with the steps.

  • Step 1 : Create the HTML file

    • Open your text editor (we can use Notepad for now) and create a new file.

  • Step 2 : Write the HTML code

    • Type the following text into the editor:

    <!DOCTYPE html> <html> <body> Hello World! </body> </html> Save the file to something like "Hello.html" in a directory that you have access to.
  • Step 3 : Open the file in your browser

    • Double click the file you just created. It should open up in your browser and look something like the image above.

  • Step 4 : Congratulations!

    • You have written and run a working web page!
