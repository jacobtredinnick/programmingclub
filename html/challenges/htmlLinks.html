Challenge - HTML Links

Task

Write 2 HTML pages that link to eachother

You should end up with 2 pages that look a bit like:

Objective

Learn how to link HTML pages with the hyperlink tag.

Tools

We will be using the HTML <a> tag to create hyperlinks.
These are known as "hyperlinks" are are usually written something like this:

<a href="Link address (URL)">Link Text</a> For example this: <!-- replace with your own school website or similar --> <a href="http://www.yourschool.co.uk">This is a link to the school website</a>

Creates a "link" like this in the browser:

This is a link to the school website

  • Step 1 : Create 2 HTML files

    • Use Notepad to create 2 new HTML files. I'll call them "file1.html" and "file2.html" but you can call them what you want.
    You'll need some HTML in the files. Look at the previous challenges if you can't remember how to write it.
    Make sure that you can open them in the browser, like in the last lesson.

  • Step 2 : Add a link from file1 to file2.

    • Make sure that the HTML for file1 contains a link like the one above, but to file2 instead of the school website.
    You can use file:// in the link address like this:

    <a href="file://...path to file1.html">This is a link to file1 (write your own link text here).</a>
  • Step 3 : Add a link from file2 to file1.
    • Make sure that the HTML for file2 contains a link to file1, as above.
