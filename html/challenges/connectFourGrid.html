Challenge - Connect Four Grid

Task

Draw the Connect Four grid.
In this challenge we will draw the grid like the one used in the Connect Four game:

The grid has 6 rows and 7 columns. For now we'll keep it simple and just draw the grid on the whole of the drawing area. We can come back later and make it look a bit better...

Tools

Use the background function to draw a blue rectangle and then use the ellipse function draw white circles with the right diameter and spacing to make the 6x7 grid for the game.

Example (Show)


int SLOT_DIAMETER=80;
int SLOT_SPACING=100;
int GRID_WIDTH=700;
int GRID_HEIGHT=600;
size(GRID_WIDTH, GRID_HEIGHT);
background(#0B0EB2); // a blue picked from Tools > Color Selector

int slotSpacing = 100;
int firstPos = slotSpacing / 2;
for (int i = 0; i < 7; i++) { // for each column
  for (int j = 0; j < 6; j++) { // for each row
    int x = firstPos + i*SLOT_SPACING;
    int y = firstPos + j*SLOT_SPACING;
    ellipse(x, y, SLOT_DIAMETER, SLOT_DIAMETER);    
  }
}
