Challenge - Hello 1000 Worlds

Task

Write a program that prints 1000 lines saying "Hello World 1!", "Hello World 2!", ... up to "Hello World 1000!": 


Hello World 1!
Hello World 2!
Hello World 3!
...
Hello World 1000!

Objective

Learn about loops and variables.

Now that you've recovered from the excitement of writing your first program, let's just quickly do it 1000 more times!
We could do this using 1000 println statements with a different number in each: 


println("Hello World 1!");
println("Hello World 2!");
println("Hello World 3!");
...
println("Hello World 1000!");
But it is much less work to do this with loops and variables...

Tools

  • A comment is a line starting with // and is ignored:
    • 
// We will use comments to explain some of the tools and examples, directly in the code...
  • A for loop to repeat other code. For example:
    • 
for (int i=1; i<=1000; i++) {
   // This loop will repeat the things between the { and the }, 1000 times, read on to see how this works...
}
    The for loop contains three parts:
  • An initialization expression declares an integer variable i to keep count, starting at 1:
    • 
for (int i=1; i<=1000; i++) {
}
  • A termination expression stops the loop when the expression is false. Here it stops if i is not less than or equal to (<=) 1000:
    • 
for (int i=1; i<=1000; i++) {
}
  • An increment expression modifies the loop variable usually by incrementing it, here using the increment operator ++ to add 1: 
    
for (int i=1; i<=1000; i++) {
   // "i++" is the same as "i = i + 1" (i++ is much quicker to type though!)
   // Adding 1 to i each time means that the loop will run 1000 times, stopping when i is 1001 because "i <= 1000" is false
}
  • A String concatenation is used to join multiple strings or other variables into one string, using the + operator:
    • 
int i = 1;
println("Hello World " + i);

    Output:

    
Hello World 1

    Example (Show)

    
// loop 1000 times
for (int i=1; i<=1000; i++) {
   // concatenate a string and print it (notice that we can concatenate more than 2 things in one statement)
   println("Hello World " + i + "!");
}
