Challenge - Firework

Objective

Use the current system time in an animation sequence

Task

  • Easy: Set off a green firework
  • Medium: Set off ten fireworks, one per second, each with a different color and start position
  • Hard: Make the fireworks launch in an arc, instead of a straight line, to look more realistic

    • Tools

    The starter program animates a single, red firework.

    Easy Challenge hints

    Change the color of the firework, where it is declared in the setup() function:

    
void setup()
{
  size(SCREEN_WIDTH, SCREEN_HEIGHT);
  smooth();
  
  firework = new Firework(400, 1, 3, color(255,0,0));
}
    Remember that the color code is an RGB value where RGB means "Red Green Blue".

    Medium Challenge hints

    Create some more fireworks in the setup function above and don't forget to draw them all in the draw() function.

    The comments in the code for Firework tell you have to create one:

    
  // Create  firework
  // x starting position: Must be between 0 and SCREEN_WIDTH
  // lightTime: Time in seconds at which the fuse is lit
  // fuseTime: Time in seconds for which the fuse burns
  // explosion: Color of the firework explosion
  Firework(int x, int lightTime, int fuseTime, color explosionColor)

    You could use random locations for the start position and color, but set either the lightTime or the fuseTime to set off one per second.

    Hard Challenge hints

    You will need to adjust x as well as y while the firework is launching. Perhaps you could use sin and cos functions if you know the maths...
