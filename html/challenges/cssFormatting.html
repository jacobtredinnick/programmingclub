Challenge - CSS Formatting

Task

Write an HTML page that uses CSS (Cascading Style Sheets) to do the formatting.

Objective

Learn how to use Cascading Style Sheets (CSS)
You can do lots of cool stuff with CSS - Make your pages look amazing!
Javascript can be used to add/change CSS codes on a page. We'll be learning how to do this in a couple of lessons time.

Introduction

HTML (HyperText Markup Language) was never intended to handle the formatting of a document.

It was designed to "mark-up" the document structure. E.g...

<h1>This is a heading</h1> <p>This is a paragraph.</p>

The recommended way of formatting our documents is to use CSS (or Cascading Style Sheets)

CSS defines how our HTML elements are displayed. For example...

this text is big and green!

This is the CSS that makes it happen...

<p style="color:green;font-size:20px">this text is big and green!</p>

The style attribute is added to the

<p> tag and it holds the CSS properties & values that define how it looks:

color is the first property and green is its value.

font-size is the second property and 20px is its value (px means the number of "pixels" or dots on the screen).
