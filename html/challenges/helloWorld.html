Challenge - Hello World

Task

Write a program that writes "Hello World!" to the screen: 


Hello World!

Objective

Write your first program!

Tools

"Hello World" is the traditional first step to learn any programming language.
Since this is the first challenge, you need all of the steps instead of only the tools. Just follow along and make sure you see something like the pictures below.

  • Step 1 : Start Processing

    • Go to the setup section if you are unable to find the "Processing.exe". When you have started Processing, you should see something like:

  • Step 2 : Write the program

    • Type the following text into the editor:

    
println("Hello World!");

    And you should have something like:

  • Step 3 : Run the program

    • Press the little triangle button in the top left of the Processing window to run your program.

    If you didn't make any mistakes, a little grey window will pop up. Ignore this and look at the output of your program (circled in the picture below):

  • Step 4 : Congratulations!

    • You have written and run a working program!

    Now save your program for later by doing "File > Save" and creating a new directory "helloWorld", somewhere you can find it next time.

    These are the programming tools that we have just used:

  • A String is a sequence of characters inside double quotes.
  • The println function writes to the console.
  • The syntax has to be exactly right or the computer will not understand. Notice the parentheses ( () ) and semicolon ( ; ) characters above.

    • Example (Show)

    
println("Hello World!");
