Challenge - Noughts And Crosses - Human Move

Task

Draw a "nought" in the first square clicked with the mouse

There are 2 stages to this challenge:

  • Highlight the square when the mouse is hovered over it:

  • Draw the nought when the mouse is clicked:

    • Objective

    Use the mouse to draw when clicked

    Tools

  • The boolean type can have only 2 possible values: true or false:

    • 
boolean humanMove = true;
  • The if statement is used to execute code if a condition is true:

    • 
// if the human is moving, highlight the square currently containing the mouse
if (humanMove) {
  drawPreviewFromMousePosition();
}
  • The else statement is used with if to execute code if a condition is not true:

    • 
if (c < SCREEN_WIDTH/3) {
  return 0;
}
else if (c < 2*SCREEN_WIDTH/3) {
  return SCREEN_WIDTH/3;
}
else {
  return 2*SCREEN_WIDTH/3;
}
  • The mouseClicked function is called by Processing when the mouse is clicked:

    • 
void mouseClicked() {
  if (humanMove) {
    // set the nought variables to the topleft corner of the square containing the mouse
    noughtX = mouseSquareCorner(mouseX);
    noughtY = mouseSquareCorner(mouseY);
    
    // and now it will be the computer's turn next...
    humanMove = false;
  }
}
