Challenge - Firework Display

Objective

Add a new type to describe multiple fireworks

Task

  • Easy: Set off a display of 20 random fireworks
  • Medium: Vary the explosion height and radius
  • Hard: Make the diaplay carry on forever

    • Tools

    The starter program animates a random display of 10 fireworks.

    This example is the same as the medium solution to the Firework challenge, except that we have replaced lots of variables:

    Easy Challenge hints

    Pass a different number to the FireworkDisplay object that is created in the setup() function:

    
void setup()
{
  size(SCREEN_WIDTH, SCREEN_HEIGHT);
  smooth();

  // create a display of 10 random fireworks
  fireworkDisplay = randomFireworkDisplay(10);
}

    Medium Challenge hints

    Try to follow the code to see how the fireworks are animated. The explosion currently happens at a fixed height and radius:

    
  int explosionRadius = 150;
  int explosionHeight = SCREEN_HEIGHT/3;

    Add new parameters to the Firework constructor like this, to set them on that particular Firework "object":

    
	  Firework(float x, int lightTime, int fuseTime, color explosionColor, int explosionHeight, int explosionRadius) {
	  ...
	  
	  this.explosionHeight = explosionHeight;
	  this.explosionRadius = explosionRadius;

    Set these new parameters from the calling code (more random values perhaps?) and see what happens...

    Hard Challenge hints

    Add a method to to the Firework type that let's us know that it is exploded: 
    
boolean isExploded() ...
    In the draw() method check for exploded fireworks in the display and replace them with new ones.
