Challenge - Snowflake Outline

Objective

Draw a snowflake outline using trigonometry functions

  • Easy: Start with a circle

  • Medium: Draw some long lines

  • Hard: Draw some short lines between the long lines

    • Tools

    The starter program defines a Snowflake type. You will have to draw it.

    Easy Challenge hints

    Draw a circle in the Snowflake.draw() method using the ellipse() function with the data contained in the snowflake: x, y, innerRadius: 
    
class Snowflake {
  // configurable
  float x;
  float y;
  float innerRadius;
  ...  
  void draw()
  {
    // get ready to draw
    stroke(flakeColor);
    strokeWeight(innerRadius/10);
    noFill();
  
    // draw the inner circle using the ellipse function
    ... you will have to call the ellipse() function here with the right parameters ...
  }

    Medium Challenge hints

    Draw six long lines and six small lines around edge of the circle: To do this it will be easiest to use some trigonomery (sorry if you've not yet done this in maths!):

    Think of the snowflake outline contained within various circles. The points on the edges of the circles are based on the sin or cos functions multiplied by the radius of the circle you are thinking about...
    You will need a loop something like this in the Snowflake.draw() method: 
    
    // draw the 6 long lines
    for (int i = 0; i < 6; ++i) {
      // work out where to draw the line
      // who has done trigonometry in maths then?
      float r = i * TWO_PI/6; // radians - need to use these for sin & cos to work properly, a full circle is TWO_PI radians
      float s = sin(r);
      float c = cos(r);
      float x0 = ...x positon on inner snowflake circle (innerRadius)...
      float y0 = ...y positon on inner snowflake circle (innerRadius)...
      float x1 = ...x positon on outer snowflake circle (outerRadius)...
      float y1 = ...y positon on outer snowflake circle (outerRadius is already stored)...
      
      // draw the line
      line(x0, y0, x1, y1);
    }

    Hard Challenge hints

    This is not that hard if you can do the medium challenge. Just some more trigonometry with slightly different angles.
