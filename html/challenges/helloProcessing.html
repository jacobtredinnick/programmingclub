Challenge - Hello Processing

Task

Write a program that shows "Hello Processing" in a window like this:

Objective

Draw your first graphics

We will be learning most of our programming using simple graphics and Processing helps make this easy.

Tools

  • The sketch window is the little grey window that we have ignored up till now. This is where we will draw graphics:

  • The size() function changes the size of the sketch window:
    • 
// make the sketch window 600 pixels wide by 400 pixels high
// default is 100x100
size(600,400);

  • The background() function changes the background colour of the sketch window:
    • 
// make the sketch window background black (0 means black here)
// Follow the colour tutorial to learn more about color codes
background(0);

  • The rect() function draws a rectangle:
    • 
// draw a rectangle with top left corner at (10,10), width 50, height 100
// computer graphics start with (0,0) at the top left (try a few different numbers and see what happens)
// read the drawing tutorial to learn more about computer graphics
rect(10,10,50,100);

  • The fill() function changes the colour used to fill the next graphic that is drawn:
    • 
// set the color of the next graphic
// here it turns the rectangle red using a "red/green/blue" (RGB) color code for bright red: red=255, blue=0, green=0
fill(255,0,0);
rect(10,10,50,100);

  • The text() function draws some text:
    • 
// draw text "Some Text" at (50,50)
text("Some Text",50,50);

  • The textSize() function changes the font size of the next text drawn:
    • 
// change the font size of the next text
// here it sets the size of "Some Text" to 48
textSize(48);
text("Some Text",20,20);

    Example (Show)

    
// make the sketch window a bit bigger, 600 wide by 400 high
size(600, 400);

// make the sketch window background black, in grayscale 0 means black
background(0);

// smooth() makes everything look a bit smoother, but take a bit longer
smooth();

// draw a red rectangle
// fill() sets the color of the next graphic to be drawn, using an RGB code
// rect() draws a rectangle with specified position and size
fill(255,0,0);
rect(50,50,500,100);

// draw some text on top of the rectangle
// textSize() sets the font size for the next text to be drawn
// text() draws the text with the specified string and position
textSize(48);
text("Hello Processing!", 100, 100);
