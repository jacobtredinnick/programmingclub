Challenge - Noughts And Crosses - Grid

Task

Draw the grid for a noughts and crosses game

The grid has 2 vertical and 2 horizontal lines making 9 equal sized squares:

Objective

Start work on a game

Tools

  • The line() function draws a line from (x1,y1) to (x2,y2)

  • The * operator multiplies 2 numbers

    • Use the other functions we have already used to draw the lines and set the background color.

    Look at the solutions to previous challenges or perhaps you have saved some of your other work?

// noughtsAndCrossesGridLine - start of a noughts and crosses grid

int SCREEN_WIDTH=600; // this game assumes a square screen for now!

void setup() {
  size(SCREEN_WIDTH, SCREEN_WIDTH);
  smooth();
}

void draw() {
  // clear everything
  background(170,200,200);

  // draw the grid
  drawGrid();
}

void drawGrid() {
  strokeWeight(10); // thick line, will already be black
  int gridWidth = SCREEN_WIDTH/3; // the width of each square
  line(0, gridWidth, SCREEN_WIDTH, gridWidth); // top horizontal line
  line(0, 2*gridWidth, SCREEN_WIDTH, 2*gridWidth); // bottom horizontal line
  line(gridWidth, 0, gridWidth, SCREEN_WIDTH); // left vertical line
  line(2*gridWidth, 0, 2*gridWidth, SCREEN_WIDTH); // right vertical line
}
