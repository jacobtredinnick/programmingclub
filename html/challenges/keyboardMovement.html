Challenge - Keyboard Movement

Objective

Use keyboard input to move an object

  • Easy:Change the color, size and speed of the ball
  • Medium:Use the arrow keys to allow the ball to move the ball
  • Hard:Do something "interesting" when the ball moves

    • Tools

    The starter program is an animation of a ball that starts in the middle and sticks on the right of the screen.

    Easy Challenge hints

    Look at the code for the things that we should change, change them and see what happens...
