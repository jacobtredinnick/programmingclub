Challenge - CSS Formatting - Part 2

Introduction

CSS lets us style elements to look however we like by adding the "style" attribute, but adding it to every HTML tag is tedious and repetitive.

You can define your styles once and then apply them to your HTML like this...

<html> <head> <style> /* Apply styles to HTML elements by type */ H1 {color:Orange} p {color:green;border-color:yellow} /* Apply styles to all HTML elements with the same "class" */ .big {font-size:36px} .small {font-size:8px} .red {color:red} /* Apply styles to specific HTML elements by ID */ #into {font-size:36px; color:blue} #main {font-size:8px; font-family:Arial} </style> </head> <body> <h1>My Header</h1> <p>This is a paragraph.</p> <p class="big">This is big a paragraph.</p> <p class="small">This is a small paragraph.</p> <p id="intro" >This is another big paragraph.</p> <p id="main" class="small">What size and color am I?</p> </body> </html>

See how it work by opening this page css2.html. Open the page and take a look inside (or select View>Source in your browser to see the HTML).

Use "class=name" and a style called ".name" when you want to affect more than one thing on a page.

Use "id=name" and a style called "#name" when you want to affect something specific on a page.

Task

Change the page that you used in the last exercise. Move your styles into a "style" section and use classes or id's to link them to your HTML elements

Objective

Write better pages by putting your styles into a <style> section in the <head> of the page.

Tools
