Challenge - Snowflake

Objective

Draw a snowflake using trigonometry functions

  • Medium: Add some more decoration to the long lines:

  • Hard: Customize your own snowflake(s)!

    Medium Challenge hints

    Use trigonometry to find x0 and y0 somewhere along the long line. Notice that we now have some helper functions with the trigonometry: 
    
  void draw()
  {
...
    // draw the 6 long lines
    for (int i = 0; i < 6; ++i) {
...
      x0 = getFlakeCircleX(r, outerRadius * 0.4);
      y0 = getFlakeCircleY(r, outerRadius * 0.4);
    Use trigonometry again to find x1 and y1 with + and - some small number of radians and a larger distanceFromCenter: 
    
      x0 = getFlakeCircleX(r-something, someotherthing);
      y0 = getFlakeCircleY(r-something, someotherthing);
      line(x0,y0,x1,y1);

      x0 = getFlakeCircleX(r+something, someotherthing);
      y0 = getFlakeCircleY(r+something, someotherthing);
      line(x0,y0,x1,y1);

    Hard Challenge hints

    Change some of the variables and see what happens. Change the new Snowflake(...) in the setup(). Add multiple flakes. Make the flakes float around. I still want some examples for the gallery!
